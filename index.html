RiWABuild-v2

RiWABuild is an IDE for php based Rich Web-based Application development
When you build the application from the source, make sure to have tidy.dll in the application folder.

Features

Split view
Live preview

Usage tips

IDE-----------------------------------------------
Create project to make directory fructure for RiWAs
Set server path for live preview of php files


EXPLORER------------------------------------------
Drag n drop to move files into folders


EDITOR--------------------------------------------
Tooltip on tab page shows the full path
Drag tabls to change tab order


CODER---------------------------------------------
ctrl + space = intellisense
for CSS - ":#" will open the color dialog
Save code to refresh the designer and the previwer
Drag and drop CSS/JS files on php/html code to link them
Context menu -> insert snippets (for JS)


DESIGNER------------------------------------------
double click an image file from explorer to add the image to the cursor position in the designer
